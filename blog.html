Question-1: What are the different ways to select an element in the DOM?

There are various methods to select an element in the DOM, including getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML retrieves or sets HTML content, parsing tags. innerText gets only visible text, ignoring hidden elements. textContent retrieves all text, including hidden elements, without parsing HTML. textContent is faster than innerText.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements, even if they are added dynamically. Instead of adding event listeners to multiple child elements, we attach a single event listener to their common parent and use event bubbling to detect the target element.

Question-4: What is event bubbling in the DOM?

Bubbling in the DOM is an event propagation method. It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element using document.createElement(). Add it using appendChild(), append(), prepend(), or insertBefore(). Remove it with remove() or removeChild(). By using these methods, we can dynamically manage elements in the DOM efficiently.
